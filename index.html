🐍 Code mẫu — Echo Bot

# bot.py — Bot Echo dùng python-telegram-bot v20+
from telegram import Update
from telegram.ext import (
    Application, CommandHandler,
    MessageHandler, filters, ContextTypes
)

TOKEN = "YOUR_BOT_TOKEN_HERE"

async def start(update: Update, context: ContextTypes.DEFAULT_TYPE):
    await update.message.reply_text(
        "👋 Chào mừng đến với Phúc Bot!\nGõ gì đó đi, tôi sẽ nhắc lại!"
    )

async def echo(update: Update, context: ContextTypes.DEFAULT_TYPE):
    text = update.message.text
    await update.message.reply_text(f"Bạn nói: {text}")

def main():
    app = Application.builder().token(TOKEN).build()
    app.add_handler(CommandHandler("start", start))
    app.add_handler(MessageHandler(filters.TEXT & ~filters.COMMAND, echo))
    print("🤖 Bot đang chạy... Nhấn Ctrl+C để dừng.")
    app.run_polling()

if __name__ == "__main__":
    main()

Giải thích siêu nhanh: Application là "trái tim" bot. CommandHandler bắt lệnh /start. MessageHandler bắt mọi tin nhắn văn bản. run_polling() giúp bot lắng nghe liên tục từ máy chủ Telegram.

🚀 Nâng cao — Bot AI Trả lời tự động (OpenAI)

# bot_ai.py — Tích hợp AI vào Telegram
import openai
from telegram.ext import Application, MessageHandler, filters

TOKEN = "YOUR_TELEGRAM_TOKEN"
OPENAI_KEY = "YOUR_OPENAI_KEY"
openai.api_key = OPENAI_KEY

async def chat_ai(update, context):
    user_msg = update.message.text
    res = openai.ChatCompletion.create(
        model="gpt-3.5-turbo",
        messages=[{"role":"user", "content":user_msg}]
    )
    reply = res.choices[0].message.content
    await update.message.reply_text(reply)

app = Application.builder().token(TOKEN).build()
app.add_handler(MessageHandler(filters.TEXT & ~filters.COMMAND, chat_ai))
app.run_polling()

Thay 2 key, bạn đã có trợ lý AI ChatGPT ngay trong Telegram. Cực kỳ phù hợp để làm tư vấn khách hàng tự động hoặc bot cá nhân thông minh.

Debug = Học

Đừng sợ lỗi. Mỗi lần fix bug là não bạn đang nạp thêm "kinh nghiệm thực chiến" cực kỳ quý giá.

So sánh bài toán "Tính diện tích hình tròn" qua 3 ngôn ngữ:
🅒 Ngôn ngữ C
#include <stdio.h>
#define PI 3.14159

int main() {
    float r = 5.0;
    float s = PI * r * r;
    printf("S = %.2f", s);
    return 0;
}
🅒🅒 C++
#include <iostream>
using namespace std;

int main() {
    const float PI = 3.14159;
    float r = 5.0;
    cout << "S = " << PI * r * r;
    return 0;
}
🐍 Python
import math

r = 5.0
s = math.pi * r ** 2
print(f"S = {s:.2f}")

📌 Giải thích bản chất cốt lõi

  • C gần phần cứng nhất. Bạn tự quản lý kiểu dữ liệu, dùng & cho scanf, kết thúc bằng ;. Chạy cực nhanh, nhẹ, phù hợp nhúng & hệ thống.
  • C++ kế thừa C nhưng thêm hướng đối tượng. cout/cin tự nhận diện kiểu dữ liệu. Dùng const thay #define an toàn hơn.
  • Python bỏ ký tự rườm rà, dùng thụt lề phân khối. input() luôn trả chuỗi nên cần ép kiểu. Code ngắn, dễ đọc, vua của AI / Data / Web.
  • Mẹo viết code mượt: Đặt tên biến có nghĩa (vd: radius thay r), comment giải thích "tại sao" (không phải "là gì"), chia nhỏ hàm (1 hàm 1 việc).
