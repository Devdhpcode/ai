🤖 Xây dựng Bot Telegram với Python

📋 Các bước triển khai

1

Mở Telegram → Tìm @BotFather → Gõ /newbot → Đặt tên → Nhận API Token (dạng 123456:ABC-DEF...).

2

Cài thư viện: pip install python-telegram-bot (hoặc dùng telepot, aiogram).

3

Copy code mẫu bên phải → Thay TOKEN của bạn vào → Chạy file bot.py.

4

Mở chat với bot, thử gõ /start và hello để test!

🐍 Code mẫu - Echo Bot

# bot.py - Bot Telegram Echo bằng python-telegram-bot v20+
from telegram import Update
from telegram.ext import Application, CommandHandler, MessageHandler, filters, ContextTypes

# 1. Dán TOKEN của bạn vào đây
TOKEN = "YOUR_BOT_TOKEN_HERE"

# 2. Hàm xử lý lệnh /start
async def start(update: Update, context: ContextTypes.DEFAULT_TYPE):
    await update.message.reply_text(
        "Chào mừng đến với Phúc Bot! 👋\nGõ gì đó đi, tôi sẽ nhắc lại!"
    )

# 3. Hàm echo: nhắc lại tin nhắn user
async def echo(update: Update, context: ContextTypes.DEFAULT_TYPE):
    user_text = update.message.text
    await update.message.reply_text(f"Bạn vừa nói: {user_text}")

# 4. Hàm main - điểm khởi đầu
def main():
    # Tạo ứng dụng bot
    app = Application.builder().token(TOKEN).build()

    # Đăng ký handler
    app.add_handler(CommandHandler("start", start))
    app.add_handler(MessageHandler(filters.TEXT & ~filters.COMMAND, echo))

    # Chạy bot (polling)
    print("🤖 Bot đang chạy... Nhấn Ctrl+C để dừng.")
    app.run_polling()

# 5. Chạy
if __name__ == "__main__":
    main()

Giải thích nhanh: Application là trái tim bot. CommandHandler bắt lệnh /start. MessageHandler bắt mọi tin nhắn văn bản. run_polling() giúp bot "lắng nghe" liên tục từ Telegram.

🚀 Nâng cao - Bot AI Trả lời tự động

# bot_ai.py - Tích hợp OpenAI API vào Telegram Bot
import openai
from telegram.ext import Application, MessageHandler, filters

TOKEN = "YOUR_TELEGRAM_TOKEN"
OPENAI_KEY = "YOUR_OPENAI_API_KEY"
openai.api_key = OPENAI_KEY

async def chat_ai(update, context):
    user_msg = update.message.text
    # Gọi OpenAI API
    response = openai.ChatCompletion.create(
        model="gpt-3.5-turbo",
        messages=[{"role": "user", "content": user_msg}]
    )
    reply = response.choices[0].message.content
    await update.message.reply_text(reply)

app = Application.builder().token(TOKEN).build()
app.add_handler(MessageHandler(filters.TEXT & ~filters.COMMAND, chat_ai))
app.run_polling()

Chỉ cần thay 2 key, bạn đã có bot AI như ChatGPT ngay trong Telegram. Tuyệt vời để làm trợ lý ảo hoặc tư vấn khách hàng tự động!

📌 Giải thích bản chất cốt lõi

  • C là ngôn ngữ "nguyên thủy", gần phần cứng nhất. Bạn phải tự quản lý kiểu dữ liệu, dùng & cho scanf, và kết thúc bằng ;. Ưu điểm: chạy cực nhanh, nhẹ.
  • C++ kế thừa C nhưng thêm hướng đối tượng. cout/cin tự động nhận diện kiểu dữ liệu. Dùng const thay #define để an toàn kiểu.
  • Python bỏ hết ký tự rườm rà, dùng thụt lề để phân khối. input() luôn trả về chuỗi nên phải ép kiểu float(). Code ngắn, dễ đọc, phù hợp AI / Web.
  • Mẹo viết code mượt: Đặt tên biến có nghĩa, comment giải thích "tại sao", chia nhỏ hàm (mỗi hàm 1 việc), tuân thủ quy ước đặt tên chuẩn.
